GARIMPO

Caça bandas por gênero (emo, screamo, crunkcore...) e mostra ouvintes pra você achar pérolas obscuras. Dados via Last.fm — não existe API pública dos "ouvintes mensais" do Spotify, então uso ouvintes totais do Last.fm (ótimo pra ranquear do menor pro maior).

1 · chave de API (grátis)

Crie em last.fm/api/account/create (leva 1 min). A chave fica só nesta aba.

2 · qual cena você quer garimpar?

Com 2 tags ele cruza as cenas e mostra só bandas que pertencem às duas. Os botões abaixo preenchem o campo que você clicou por último.

Não sabe quais tags existem?

3 · como garimpar

Varredura multi-página: escolha "1–10" + ordene por menos ouvintes pra montar um pool grande e pinçar as menores. Pra mirar ~500 ouvintes, comece numa página inicial mais funda (ex: 20–40). Analiso no máx. 250 bandas por garimpo pra não estourar o limite do Last.fm.

4 · bandas parecidas (algoritmo do Last.fm)

Usa o artist.getSimilar do Last.fm. Lista as parecidas com o % de similaridade + ouvintes. Cada resultado (aqui e no garimpo) ganha um botão ↗ parecidas pra encadear a descoberta.
